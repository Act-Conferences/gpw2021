Über Perl und RakuAbout Perl and Raku

Perl und Raku sind Open-Source-Programmiersprachen: Perl and Raku are Open Source programming languages:

  • Perl 5 wird seit 1987 weiterentwickelt, 2020 wird die Version 5.32 freigegeben. Perl 5 ist eine der meistgenutzten Programmiersprachen, läuft auf Plattformen vom Handy bis zum Mainframe und bietet mit dem "Comprehensive Perl Archive Network" (CPAN) über 35000 Erweiterungen an. Perl 5 is being developed since 1987, version 5.32 will be released in 2020. Perl 5 is one of the most popular languages and can be used on almost every platform between a mobile phone and a mainframe system. The "Comprehensive Perl Archive Network" (CPAN) offers more than 35.000 extensions.
  • Raku ist eine damit verwandte Sprache, die viele Elemente aktueller Software-Architekturen direkt in die Sprachbeschreibung aufgenommen hat und dafür bewusst auf Kompatibilität mit Perl 5 verzichtet. Raku is a related language which has adopted many paradigms of current software architectures directly into the language, intentionally giving up source compatibility to Perl 5.

Die Perl Foundation ist eine gemeinnützige Organisation, die sich um die Weiterentwicklung und Unterstützung der Gemeinschaft der Benutzer einsetzt. The Perl Foundation is a non-profit organisation dedicated to the advancement of the Perl programmig languages and to supporting the user community.

Perl Mongers sind lokale Benutzergruppen. In Deutschland veranstaltet der Frankfurter Perlmongers e.V. den Deutschen Perl/Raku-Workshop, unterstützt von jährlich wechselnden lokalen Organisationen wie den Leipzig Perl Mongers im Jahr 2021. Perl Mongers are local user groups. In Germany, the Frankfurt Perlmongers e.V. is organizing the annual German Perl/Raku Workshop, supported by changing local user groups like the Leipzig Perl Mongers in 2021.
